Footwear Program

Create a class Footwear with the following private attributes:

footwearId - int
footwearName - String
footwearType - String
price - int

The above attributes should be private.

Write:

Getter methods
Setter methods
Parameterized constructor

as required.

Create a class FootwearProgram with the main() method.

Implement the following two static methods:

1. getCountByType()

This method will take:

An array of Footwear objects
A String parameter representing the footwear type

The method should return the count of footwears whose type matches the given footwear type.

If no footwear with the given type is found, return 0.

2. getSecondHighestPriceByBrand()

This method will take:

An array of Footwear objects
A String parameter representing the footwear brand name

The method should return the Footwear object having the second highest price among the footwears whose brand name matches the given input.

If no footwear with the given brand is found, return null.

Note
No two footwear objects will have the same footwearId.
All searches should be case-insensitive.
The above methods should be called from the main() method.
Output Conditions
For getCountByType()

If the returned count is greater than 0, print the count.

Otherwise print:

Footwear not available
For getSecondHighestPriceByBrand()

If a valid Footwear object is returned, print:

footwearId
footwearName
price

Otherwise print:

Brand not available
